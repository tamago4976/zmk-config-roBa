# zmk-config-roBa

roBa 用 ZMK キーマップの早見表です。  
設定のベースは [mozumasu/zmk-config-roBa](https://github.com/mozumasu/zmk-config-roBa) です。

以前のカスタムレイアウトはブランチ `backup/custom-layout-20260529` に保存しています。

![キーマップ図](keymap-drawer/roBa.svg)

---

## 覚え方のコツ（3つだけ）

| やりたいこと | 操作 |
|-------------|------|
| **記号・数字** | 左親指 **Space を押しっぱなし**（NUM レイヤー） |
| **英数 / かな** | 親指 **英数**・**かな** をタップ |
| **Fキー** | **`;` を押しっぱなし**（FUNCTION レイヤー） |

`trans`（透明）のキーは、そのレイヤーでは何も割り当てず **下のレイヤー（通常）のまま** 動きます。

---

## 親指キー（いちばんよく使う）

| キー（位置） | タップ | ホールド |
|-------------|--------|----------|
| **Space**（左内側） | スペース | **NUM**（数字・記号） |
| **英数**（左中央寄り） | 英数 | **ARROW**（矢印など） |
| **かな**（右内側） | かな | **FN**（音量・明るさ） |
| **`;`**（左手 N の右・内側） | `;` | **FUNCTION**（F1〜F12） |

---

## 通常レイヤー（何も押さないとき）

```
左手                          右手
┌───┬───┬───┬───┬───┐       ┌───┬───┬───┬───┬───┐
│ Q │ W │ E │ R │ T │       │ Y │ U │ I*│ O │ P │
├───┼───┼───┼───┼───┤       ├───┼───┼───┼───┼───┤
│ A │ S │ D │ F │ G │ ⌘⇧S │- │ H │ J │ K │ L │ ' │
├───┼───┼───┼───┼───┤       ├───┼───┼───┼───┼───┤
│⇧Z*│ X │ C │ V │ B │ 左クリ│;* │ N │ M │ , │ . │⇧/│
├───┼───┼───┼───┼───┤       ├───┼───┼───┼───┼───┤
│⌘ │⌥ │Tab│⌃* │Spc*│英数*│ ⌘ │かな*│           \ │
└───┴───┴───┴───┴───┘       └───┴───┴───────────┘

* Z     … タップ Z / ホールド Shift
* /     … タップ / / ホールド Shift
* ⌃     … タップ Esc / ホールド Ctrl
* Spc   … タップ Space / ホールド NUM
* 英数  … タップ 英数 / ホールド ARROW
* かな  … タップ かな / ホールド FN
* ;     … タップ ; / ホールド FUNCTION
* I     … タップ i / ホールド Bluetooth
```

### 通常レイヤーだけで打てる記号

| キー | 出力 |
|------|------|
| G の右（内側） | `-` |
| L の右 | `'` |
| M の右 | `,` `.` |
| `/`（タップ） | `/` |
| 右下 | `\` |

## NUM レイヤー（Space ホールド中）

**Space を押したまま** 下表のキーを押します。  
（Web の Keymap Editor では `JP_*` が表示されないことがありますが、実機では問題なく出ます。）

### 左手

| 位置 | キー | 出力 |
|------|------|------|
| 上段 | 左端 | `-` |
| 上段 | 続き | `7` `8` `9` |
| 上段 | 右端 | `+` |
| 中段 | 左端 | `/` |
| 中段 | 続き | `4` `5` `6` |
| 中段 | 右端 | `*` |
| 下段 | 続き | `1` `2` `3` |
| 下段 | | `.` |
| 下段 | 右 | `=` |
| 下段 | 左端（⇧付き） | `0`（Shift ホールド付き） |
| 親指列 | すべて | 下のレイヤー透過（通常キーのまま） |

### 右手（記号はここがメイン）

| 位置 | 出力 |
|------|------|
| 上段 左→右 | `^` `&` `~` `(` `)` |
| 中段 左→右 | `_` `!` `@` `#` `$` `%` |
| 下段 左→右 | `[` `]` `{` `}` `\` |
| 右下 | `\|` |

### 記号クイック参照（Space + 右手）

```
        ^  &  ~  (  )
        _  !  @  #  $  %
        [  ]  {  }  \
                    |
```

## その他のレイヤー

### FUNCTION（`;` ホールド）

| 右手 |
|------|
| F1〜F5（上段） |
| F6〜F10（中段） |
| F11（下段） |
| F12（右下） |
| F13（中段・内側） |

左手は `trans`（通常レイヤーのまま）。

### FN（`かな` ホールド）

| 右手 |
|------|
| 明るさ − / ＋ |
| ミュート |
| 音量 − / ＋ |

### ARROW（`英数` ホールド）

| 左手 | 出力 |
|------|------|
| 上段 | Esc / 前タブ / ↑ / 次タブ |
| 中段 | Home / ← / ↓ / → / End |
| 下段 | Shift + 選択範囲拡張（←→） |

### Bluetooth（`I` ホールド）

| 右手 |
|------|
| BT 0〜4 選択 |
| BT クリア / 全クリア |
| Bootloader（内側） |

### MOUSE

mozumasu 設定ではレイヤー番号のみ定義。親指などから直接 `mo` していないため、通常はこのレイヤーには入りません。

## コンボ（2キー同時・20ms 以内）

| 同時に押すキー | 出力 |
|---------------|------|
| **S + D** | Tab |
| **D + F** | Shift + Tab |
| **L + '**（L の右） | `"` |
| **C + V** | `=` |
| **J + K** | 左クリック |
| **K + L** | 右クリック |

| レイヤー | 動作 |
|----------|------|
| 通常 / ARROW | スクロール（上下） |

## よくある操作例

| やりたいこと | 操作 |
|-------------|------|
| `!` を打つ | Space ホールド + 右手中段（`!` の位置） |
| `(` `)` | Space ホールド + 右手上段の左右 |
| `@` | Space ホールド + 右手中段 |
| Tab | **S + D** 同時 |
| 英数にする | 親指 **英数** タップ |
| かなにする | 親指 **かな** タップ |
| F5 | **`;` ホールド** + 右手上段 |
